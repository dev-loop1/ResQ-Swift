## Introduction

Welcome to ResQ Swift, a project that aims to revolutionize access to ambulance services during medical emergencies. This platform addresses the critical need for timely medical assistance, potentially saving lives in the process.

---

## Problem Statement

ResQ Swift addresses the critical issue of delayed access to ambulance services during medical emergencies, which can have life-altering consequences. Many individuals struggle to find the appropriate type of ambulance for their specific emergency, locate nearby hospitals, and administer initial first aid care, especially in isolated areas. The absence of a centralized and user-friendly platform exacerbates this challenge.

---

## Product Description

ResQ Swift is a project dedicated to providing timely access to vital medical assistance in emergencies. The Ambulance Booking Website empowers users to swiftly locate and reserve the most suitable ambulance, choose nearby hospitals, and receive essential first aid guidance. Sharing specific injury information with the hospital in advance ensures they are prepared for the patient's arrival, expediting treatment. This significantly enhances the chances of positive outcomes during emergencies and, ultimately, saves lives.

Furthermore, this platform extends its benefits to individuals with auditory and vocal limitations, offering an inclusive solution for ambulance booking. A groundbreaking feature introduces the ability to transmit critical patient information, including the nature of the emergency, to the hospital in advance. This innovation is a game-changer in patient care and emergency response, dramatically reducing response times in crucial moments. It serves as a life-saving beacon, underscoring our commitment to redefining healthcare through technology-driven impact.

## Features

- Ambulance Booking: 
Users can quickly find and book the appropriate type of ambulance for their specific emergency.
- Hospital Selection: 
Users can identify nearby hospitals for timely treatment.
- First Aid Guidance: 
Provides initial first aid care instructions for individuals in isolated areas.
- Emergency Information Sharing: 
Sends crucial patient information to the hospital ahead of their arrival, reducing response times.
- Accessibility: 
Designed to assist individuals with auditory and vocal issues in booking ambulances.

## Getting Started

To begin using ResQ Swift, follow these steps:

Install Dependencies:
Hyper Terminal: Download and install Hyper Terminal for an enhanced command-line interface.
Node.js: Install Node.js to run the server-side JavaScript code.
MongoDB: Set up MongoDB to store and manage data.
MongoDB Compass: Optionally, you can install MongoDB Compass for a graphical user interface to interact with your MongoDB database.
Install npm Packages:

Use the following command to install the required npm packages:
"npm install express mongoose body-parser geo-point ejs node-notifier node-navigator node-mailer googleapis"

This command will download and install the necessary packages to run the ResQ Swift web application.

Configuration:

Launch the Application:
Start the ResQ Swift web-application by running the command node app.js in your terminal. Note that your current working directory should be same as the location where you are maintining other dependencies.

Access the Platform:
Open a web browser and navigate to http://localhost:3000 to start using ResQ Swift.

## Usage

ResQ Swift is a comprehensive platform designed to facilitate efficient access to emergency medical assistance. 
The system offers several key features:

User Registration and Login System:
Users can easily register and create an account on ResQ Swift, ensuring a secure and personalized experience.

Efficient Emergency Description and Hospital Notification:
Users, like Jane, can quickly provide detailed descriptions of their emergencies through the platform. This information is immediately sent to nearby hospitals, allowing them to prepare for the incoming patient.

First Aid Tips:
ResQ Swift provides a valuable resource in the form of first aid tips. This equips users with basic knowledge to provide preliminary care while waiting for the ambulance.

Patient Information Storage:
The platform allows users to securely store essential patient information, including allergies, medical conditions, and emergency contacts. This ensures that emergency responders have quick access to crucial data.

Booking History for Patients and Hospitals:
ResQ Swift offers a comprehensive booking history feature for both patients and hospitals. Patients can review their past bookings for record-keeping and insurance purposes, while hospitals can better manage their resources and maintain administrative records.

This use case outlines the key functionalities of ResQ Swift, showcasing how it empowers users to efficiently request emergency medical assistance and provides valuable tools for both users and hospitals
